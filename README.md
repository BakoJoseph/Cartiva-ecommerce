# Cartiva-ecommerce

A full-stack e-commerce application built with React, Node.js and Express. It includes a client store front, admin panel for product and order management, and a backend server with REST APIs

## Features

User Authentication
Product Listing,
Shopping Cart,
Admin Dashboard

## Tech Stack

React
Node.js
Express

## Installation

```bash
npm install
npm run dev
```

## Usage

1. Start the backend server by running `npm run dev` in the backend folder.
2. Start the frontend client by running `npm start` in the frontend folder.
3. Open the client in a web browser and start shopping.
